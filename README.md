# nginx + apache + php + postgresql

## Steps to start

### Clone repository
`git clone https://github.com/HIKAMURACHI/docker_hell_task.git`

### Move to folder
`cd docker_hell_task`

### Run
#### Normal
`docker-compose up`
#### Background
`docker-compose up -d`

### Test app
`In your browser go to the address: 10.100.11.11`

### Drop
#### Normal
`press ctrl + c` \
`docker-compose down`
#### Background
`docker-compose down`
